Politique de confidentialité - Yumel

Dernière mise à jour : 14 janvier 2026

Yumel respecte votre vie privée. Cette politique explique comment nous collectons et utilisons vos données.

1. Données collectées

  • Informations de base fournies par l'utilisateur (par ex. email si inscription)
  • Données d'utilisation de l'application (films consultés, listes créées)
  • Informations techniques (type d'appareil, version du système d'exploitation)

2. Utilisation des données

Les données collectées sont utilisées uniquement pour :

  • Améliorer l'expérience utilisateur
  • Identifier et corriger des bugs
  • Fournir certaines fonctionnalités (connexion, préférences, rooms sociales)
  • Synchroniser vos données entre vos appareils

3. Partage des données

Nous ne vendons ni ne partageons vos données personnelles avec des tiers, sauf dans les cas suivants :

  • Pour respecter la loi
  • Avec les fournisseurs de services nécessaires au fonctionnement de l'application (Firebase, Apple, Google)

4. Cookies et technologies similaires

Yumel n'utilise pas de cookies sur son site web. L'application mobile peut utiliser des technologies de stockage local pour améliorer votre expérience utilisateur.

5. Durée de conservation des données

Vos données sont conservées tant que votre compte est actif. Si vous supprimez votre compte, vos données personnelles seront supprimées dans les 30 jours suivant votre demande.

6. Sécurité

Nous prenons des mesures raisonnables pour protéger vos informations. Cependant, aucune méthode de transmission ou de stockage n'est totalement sécurisée, nous ne pouvons donc pas garantir une sécurité absolue.

7. Droits des utilisateurs (RGPD)

Conformément au Règlement Général sur la Protection des Données (RGPD), vous disposez des droits suivants :

  • Droit d'accès : Vous pouvez demander une copie de vos données personnelles
  • Droit de rectification : Vous pouvez corriger vos données inexactes
  • Droit à l'effacement : Vous pouvez demander la suppression de vos données
  • Droit à la portabilité : Vous pouvez recevoir vos données dans un format structuré
  • Droit d'opposition : Vous pouvez vous opposer au traitement de vos données

Pour exercer ces droits, contactez-nous à l'adresse indiquée ci-dessous.

8. Contact

Pour toute question concernant cette politique de confidentialité, vous pouvez nous contacter à : yumel.app@gmail.com
